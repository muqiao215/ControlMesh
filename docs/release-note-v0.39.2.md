# ControlMesh v0.39.2

This release republishes the packaged hotfix runtime work on top of the follow-up CI assertion fixes required for stable cross-environment testing.

## Highlights

- Keeps the packaged hotfix runtime and upgrade classification introduced in `v0.39.0`.
- Includes the CI follow-up fixes from `v0.39.1`.
- Stabilizes release-line tests across root and non-root environments.

## Verification

- GitHub Actions `CI` workflow on `main`
- GitHub Actions `Publish to PyPI` workflow on tag `v0.39.2`

## Upgrade Notes

- `v0.39.0` and `v0.39.1` remain superseded release candidates.
- Use `v0.39.2` as the first corrected public release for this hotfix-packaging line.
